---
title: Extract Document
api: POST /dastavez/extract
description: "Extract structured data from Indian documents"
---

## Request

<ParamField body="document_type" type="string" required>
  Type of document:
  - `aadhaar` - Aadhaar card
  - `pan` - PAN card
  - `voter_id` - Voter ID card
  - `passport` - Indian passport
  - `driving_license` - Driving license
  - `bank_statement` - Bank account statement
  - `itr` - Income Tax Return
  - `form_16` - Form 16
  - `gst_invoice` - GST invoice
  - `salary_slip` - Salary slip
  - `auto` - Auto-detect document type
</ParamField>

<ParamField body="file" type="file">
  Document file (multipart upload). Supports PDF, PNG, JPG, TIFF.
</ParamField>

<ParamField body="file_url" type="string">
  URL to document file. Either `file` or `file_url` required.
</ParamField>

<ParamField body="options" type="object">
  Extraction options.

  <Expandable title="options properties">
    <ParamField body="mask_sensitive" type="boolean" default="false">
      Mask sensitive fields (Aadhaar number, account numbers).
    </ParamField>
    <ParamField body="extract_photo" type="boolean" default="false">
      Extract photo from ID documents.
    </ParamField>
    <ParamField body="validate" type="boolean" default="true">
      Validate checksums and formats.
    </ParamField>
    <ParamField body="language_hint" type="string">
      Hint for document language: `hi`, `en`, `ta`, etc.
    </ParamField>
    <ParamField body="enhance_image" type="boolean" default="true">
      Auto-enhance low-quality images.
    </ParamField>
  </Expandable>
</ParamField>

<ParamField body="webhook_url" type="string">
  URL for completion webhook.
</ParamField>

<RequestExample>
```bash cURL
curl -X POST https://api.rotavision.com/v1/dastavez/extract \
  -H "Authorization: Bearer rv_live_..." \
  -H "Content-Type: application/json" \
  -d '{
    "document_type": "aadhaar",
    "file_url": "https://storage.example.com/docs/aadhaar-123.pdf",
    "options": {
      "mask_sensitive": true,
      "extract_photo": true,
      "validate": true
    }
  }'
```

```python Python
from rotavision import Rotavision

client = Rotavision()

# From URL
result = client.dastavez.extract(
    document_type="aadhaar",
    file_url="https://storage.example.com/docs/aadhaar-123.pdf",
    options={
        "mask_sensitive": True,
        "extract_photo": True
    }
)

# From file
with open("aadhaar.pdf", "rb") as f:
    result = client.dastavez.extract(
        document_type="aadhaar",
        file=f,
        options={"mask_sensitive": True}
    )

print(f"Name: {result.fields['name']}")
print(f"Name (English): {result.fields['name_english']}")
```

