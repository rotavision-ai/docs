---
title: Create Agent
api: POST /dastavez/agents
description: "Create a browser automation agent"
---

## Request

<ParamField body="name" type="string" required>
  Human-readable name for the agent.
</ParamField>

<ParamField body="description" type="string">
  Description of the agent's purpose.
</ParamField>

<ParamField body="capabilities" type="array" required>
  Agent capabilities:
  - `navigate` - Navigate to URLs
  - `click` - Click elements
  - `type` - Type text into inputs
  - `extract` - Extract data from pages
  - `screenshot` - Take screenshots
  - `download` - Download files
  - `login` - Handle authentication
  - `captcha` - Solve simple captchas
</ParamField>

<ParamField body="credentials" type="array">
  Stored credentials for authentication.

  <Expandable title="credential properties">
    <ParamField body="key" type="string" required>
      Unique key to reference this credential.
    </ParamField>
    <ParamField body="type" type="string" required>
      Credential type: `basic`, `form`, `oauth`.
    </ParamField>
    <ParamField body="username" type="string">
      Username or email.
    </ParamField>
    <ParamField body="password" type="string">
      Password (encrypted at rest).
    </ParamField>
  </Expandable>
</ParamField>

<ParamField body="config" type="object">
  Agent configuration.

  <Expandable title="config properties">
    <ParamField body="timeout_ms" type="integer" default="30000">
      Default timeout for actions.
    </ParamField>
    <ParamField body="viewport" type="object">
      Browser viewport size.
    </ParamField>
    <ParamField body="user_agent" type="string">
      Custom user agent string.
    </ParamField>
    <ParamField body="proxy" type="string">
      Proxy server URL.
    </ParamField>
  </Expandable>
</ParamField>

```python Python
from rotavision import Rotavision

client = Rotavision()

agent = client.dastavez.create_agent(
    name="GST Portal Agent",
    description="Fetches GST returns and invoices from GST portal",
    capabilities=["navigate", "login", "extract", "download", "screenshot"],
    credentials=[
        {
            "key": "gst_portal",
            "type": "form",
            "username": "gstin@company.com",
            "password": "encrypted_password"
        }
    ],
    config={
        "timeout_ms": 60000,
        "viewport": {"width": 1920, "height": 1080}
    }
)

print(f"Agent ID: {agent.id}")
```
