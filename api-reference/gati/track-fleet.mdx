---
title: Track Fleet
api: POST /gati/fleet/track
description: "Update vehicle locations for real-time tracking"
---

## Request

<ParamField body="updates" type="array" required>
  Vehicle location updates.

  <Expandable title="update properties">
    <ParamField body="vehicle_id" type="string" required>
      Vehicle identifier.
    </ParamField>
    <ParamField body="location" type="object" required>
      Current location with lat/lng.
    </ParamField>
    <ParamField body="speed_kmh" type="number">
      Current speed.
    </ParamField>
    <ParamField body="heading" type="number">
      Heading in degrees (0-360).
    </ParamField>
    <ParamField body="timestamp" type="string">
      ISO 8601 timestamp.
    </ParamField>
    <ParamField body="status" type="string">
      Vehicle status: `moving`, `stopped`, `idle`.
    </ParamField>
  </Expandable>
</ParamField>

<RequestExample>
```python Python
client.gati.track_fleet(
    updates=[
        {
            "vehicle_id": "v1",
            "location": {"lat": 12.9450, "lng": 77.6100},
            "speed_kmh": 35,
            "heading": 90,
            "status": "moving"
        },
        {
            "vehicle_id": "v2",
            "location": {"lat": 12.9600, "lng": 77.7200},
            "speed_kmh": 0,
            "status": "stopped"
        }
    ]
)
```
</RequestExample>
