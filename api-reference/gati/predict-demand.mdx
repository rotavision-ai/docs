---
title: Predict Demand
api: POST /gati/demand/predict
description: "Forecast demand by location and time"
---

## Request

<ParamField body="regions" type="array" required>
  Regions to predict demand for.

  <Expandable title="region properties">
    <ParamField body="id" type="string" required>
      Region identifier.
    </ParamField>
    <ParamField body="bounds" type="object">
      Geographic bounds (north, south, east, west).
    </ParamField>
    <ParamField body="center" type="object">
      Center point with radius_km.
    </ParamField>
    <ParamField body="pincode" type="string">
      Indian pincode.
    </ParamField>
  </Expandable>
</ParamField>

<ParamField body="period" type="object" required>
  Prediction period.

  <Expandable title="period properties">
    <ParamField body="start" type="string" required>
      Start date/time (ISO 8601).
    </ParamField>
    <ParamField body="end" type="string" required>
      End date/time (ISO 8601).
    </ParamField>
    <ParamField body="granularity" type="string" default="hour">
      Time granularity: `hour`, `day`, `week`.
    </ParamField>
  </Expandable>
</ParamField>

<ParamField body="factors" type="object">
  Additional factors to consider.

  <Expandable title="factors properties">
    <ParamField body="weather" type="boolean" default="true">
      Include weather impact.
    </ParamField>
    <ParamField body="events" type="boolean" default="true">
      Include local events.
    </ParamField>
    <ParamField body="holidays" type="boolean" default="true">
      Include holiday patterns.
    </ParamField>
  </Expandable>
</ParamField>

<RequestExample>
```python Python
forecast = client.gati.predict_demand(
    regions=[
        {"id": "koramangala", "pincode": "560034"},
        {"id": "indiranagar", "pincode": "560038"},
        {"id": "whitefield", "pincode": "560066"}
    ],
    period={
        "start": "2026-02-01",
        "end": "2026-02-07",
        "granularity": "hour"
    },
    factors={
        "weather": True,
        "events": True,
        "holidays": True
    }
)

for region in forecast.predictions:
    print(f"{region.id}: Peak demand at {region.peak_hour} ({region.peak_demand} orders)")
```
</RequestExample>
